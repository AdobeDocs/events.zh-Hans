# Acrobat Sign中的Web窗体快速入门

>[!VIDEO](https://video.tv.adobe.com/v/3491645/?learn=on&enablevpops)

是否仍会来回发送电子邮件表格，或逐个记录签名？ Acrobat Sign中的Web表单允许您发布单个可签名表单，任何人都可以从您的网站、共享链接或嵌入页面访问、填写和签名而无需发送任何内容。 观看本动手网络研讨会，我们的专家将涵盖：

* 什么是Web窗体？它们在哪里闪耀？从发布数百名员工可以完成的新员工入门信息包的人力资源团队，到在网站上发布面向客户的应用程序以大规模收集签名的财务团队，等等！
* 如何创建和启动Web表单 — 观看我们有关如何创建首个Web表单的实时演练，包括：上传文档或模板、放置表单字段、配置签名者身份验证以及将表单发布为可共享URL或可嵌入代码。
* 如何管理和跟踪Web表单结果 — 实时监控提交、下载表单字段数据、编辑或禁用实时表单，并随着使用量的增长保持Web表单的组织性。

无论您是从十个人还是从一万人那里收集签名，您都可以放心地创建、发布和管理您的第一个Web表单。
