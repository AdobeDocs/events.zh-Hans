# 大规模建立数字信任：使用Acrobat Sign进行身份验证和安全

掌握Adobe的最新创新，这些创新侧重于Acrobat Sign中的身份验证、信任和合规性。 观看Acrobat Sign最近发布的版本如何通过增强的安全控制和现代身份识别功能，强化各方对每份协议的信心。 包括：

* 可信数字签名：由信任服务提供商组成的广泛全球生态系统支持的基于证书的数字签名，可确保完整性、不可否认性和长期信任。
* 全球数字身份：数字身份网关可在全球范围内提供高可信度的身份验证，包括开放银行和政府颁发的eID方案。
* 高度可靠、监管就绪的签名：符合行业标准和新法规（包括eIDAS 2.0）的高级合格电子签名。
