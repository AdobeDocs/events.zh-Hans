# 谁点击了？他们什么时候点击的？

了解客户的历程至关重要。

有关用户与您的 Web 或移动资产的互动的报告有助于您了解客户历程。Adobe [!DNL Analytics] 向您展示应用程序中每次点击的点击者、内容、原因和位置，我们将在10月21日的实时活动中深入探讨如何充分利用此产品。 我们将向您说明如何将分类功能与Activity Map结合使用，以便更好地了解用户行为并轻松绘制哪些点击导致有价值的网站活动的图表。

我们还将介绍一些提示，帮助您了解和利用归因报表。
