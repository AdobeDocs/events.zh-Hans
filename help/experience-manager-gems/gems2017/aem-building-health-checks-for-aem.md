# 构建AEM的运行状况检查{#building-health-checks-for-aem}

确定AEM实例是否正常运行并不是一项简单的任务。 本会议将深入分析操作仪表板的“运行状况报表”部分：一组现成的检查，这些检查基于我们的安全和性能建议并使用Sling运行状况检查框架开发。 您将了解如何修复一些常见的未通过运行状况检查，如何使用外部工具检查其结果，以及如何编写自己的应用程序特定检查。
