# 为AEM开发OSGi包和服务{#developing-osgi-bundles-and-services-for-aem}

在本课程中了解为AEM开发OSGi包和服务是多么容易。 此会话首先简要介绍OSGi的基本概念：捆绑包和服务。 随后，它演示了如何使用成熟注释来开发捆绑包和服务，既简单又直接。 此外，还包括使AEM开发更轻松并避免缺点和隐患的提示和技巧以及最佳实践。
