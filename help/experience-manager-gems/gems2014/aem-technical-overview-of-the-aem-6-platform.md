# AEM 6平台的技术概述{#technical-overview-of-the-aem-platform}

在本次研讨会中，Peter Klassen — 产品经理将了解以下内容：重新加载JCR存储库 — 一次性改善性能和新增可扩展性选项，AEM 6中的mongoDB（存储二进制文件的位置） — 新的DataStore选项概述，使平台操作更轻松。 本课程的目标受众是合作伙伴和客户；项目经理、架构师和管理员。
