# 更智能地捕获，在全球范围内扩展 — Workfront中的自定义Forms和本地化

在“高级企业运营(AEO)功能事件”系列的第二次会议上，我们的Adobe产品经理Khachatur Sedrakyan和Kristine Tumanyan探索了四种新的高级逻辑类型：显示、可编辑、条件格式和默认值，实现了更动态、自动化和数据精确的摄取体验。

此外，他们还引入了新的本地化增强功能，允许团队为用户输入的标签添加自定义翻译，帮助全球组织跨Workfront创建更加一致、可访问且按区域一致的体验。
