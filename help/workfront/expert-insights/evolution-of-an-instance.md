# Workfront Expert Insights — 与Jen Desmond合作开发实例的演变

欢迎访问最新的Expert Insights剧集！ 我是Cynthia Boon，您的On-Demand Workfront CSM。 您是否曾想知道，其他客户如何随着时间推移发展其Workfront实例，或者如何在不让人感到不知所措的情况下浏览更改？

在本剧中，Adobe技术成功顾问Jen Desmond令人难以置信，她将与我们分享她从首次Workfront用户到可信任专家的历程，以及关于管理更改和有意增长实例的实用建议。

>[!VIDEO](https://video.tv.adobe.com/v/3483111/?learn=on&enablevpops)

您将获得一些深思熟虑的见解，包括：

* 为什么Workfront中的更改会让人感到“沉重”，以及如何以积极的心态重新构建它
* 接受并关注在浏览更改时可以控制的内容的重要性
* 如何在更改历程中引导您的用户（而不是引导他们发生更改）
* 简化流程和了解何时需要发展的现实战略
* 在优化实例时，从全新的角度来平衡工作与价值

关于“如何减轻负担”问题的更多思考：
这可能会引起争议，但会添加到您的实施时间表中。 请勿对更改的上线日期作出过度承诺。 在您的计划中，您将尽一切可能覆盖所有需要发生的事情，但任何事情多久进行一次计划？ 你不想因为知道谁受苦而烦恼吗？ 您的用户。 然后，你的用户采用坦克，然后祝你好运。 我知道，要争取利益相关者的支持来推出一些东西是很难的，我知道，因为我曾经有过这样的对话，但现实是，花更多时间，或者以后用更糟糕的方式为它买单。

另一件事 — 别忘了沟通！
面对变化，沟通，沟通，沟通！ 一旦您掌握了更改的具体情况，请立即告知用户。 不要惊吓他们，也不要等到事情发生后才对他们进行培训。 给他们时间和空间来惹恼他们，然后支持他们，让他们做好准备，以免他们真的需要改变。

## 关于扬声器

Jen Desmond是一名高级技术成功顾问，他与客户合作以简化复杂的流程，提高采用率，并设计能够反映团队实际工作方式（而不只是他们应该如何工作）的解决方案。 她对Workfront Fusion情有独钟，乐于帮助客户了解自动化如何减少手动操作、连接系统和满怀信心地扩展其流程。 Jen始终将重点放在以下方面：哪些切实可行的措施可帮助组织更快行动、更有效地进行协作并以可持续的方式增长。
