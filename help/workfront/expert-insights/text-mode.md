# Workfront Expert Insights：文本模式与Katherine Lanning

欢迎使用最新的Expert Insights！  我是Cynthia Boon，您的“On-Demand”Workfront CSM，对于本集，我们有来自IQVIQ的Workfront文本模式Jedi、Katherine Lanning、Workfront解决方案工程师！  在这一集里，Katherine分享了她的故事，她对文本模式报告的所有事情的热情，并演示了她如何构建自己的“理智管理”仪表板！

文本模式可能具有挑战性，但回报却惊人。  记住凯瑟琳的忠告！

* “如果你能看到它，你可以改变它。”  — 在标准报表中翻转到文本模式以查看如何构建列。
* 通过修修补补来学习。  — 添加字段，切换到文本模式，并进行试验以了解驱动数据的因素。
* 使用纯语言。  — 标题列是问题列或清除语句，以便非管理员可以轻松解释。
* 添加描述。  — 在文本模式下自定义悬停文本以说明数据和指导用户。
* 使可视化图表易于访问。  — 不要仅依赖颜色（例如，红色/绿色）。 使用不同的图标或标签。
* 利用社区。  — 依靠Experience League、Cookbook、API Explorer和同事。 不要重新发明轮子。
* 会一直这样，但会休息一下。  — 如果卡住，请合上笔记本电脑，散步，然后重新睁大眼睛。
