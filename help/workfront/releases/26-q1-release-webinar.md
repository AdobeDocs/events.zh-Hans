# Adobe Workfront 2026年第一季度发布网络研讨会

Workfront第一季度版本于2026年1月15日部署。 此网络研讨会是与Adobe产品管理团队合作主办的，它重点介绍了New Requesting Experience中的功能、 Canvas Dashboards中的更新、主页提及小组件的改进以及新的Planning和Fusion功能。

>[!VIDEO](https://video.tv.adobe.com/v/3481817/?learn=on&enablevpops)

查看Experience League上的[发行说明](https://experienceleague.adobe.com/zh-hans/docs/workfront/using/product-announcements/product-releases/release-26-q1/26-q1-release-overview)，因为网络研讨会期间并未涵盖所有功能。

每月都会添加新活动，因此请务必查看[Experience League活动页面](https://experienceleague.adobe.com/zh-hans/events)以了解最新会话。
