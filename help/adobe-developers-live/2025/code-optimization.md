# 引擎内部：使用Adobe Experience Manager Sites Optimizer优化代码

了解Adobe Experience Manager Sites Optimizer如何通过检测冗余脚本和繁重的DOM结构来帮助开发人员提高代码质量。 此会议演示了如何使用该工具来改进Core Web Vitals、减少技术债务并提高性能。 了解如何在您的工作流中无缝应用推荐，以实现更快的修复、更强的SEO和更顺畅的用户体验。

>[!VIDEO](https://video.tv.adobe.com/v/3477727/?captions=chi_hans&learn=on&enablevpops)

此会议 — 引擎内部：使用Adobe Experience Manager Sites Optimizer进行代码优化 — 特别演示了Deck Reyes ，介绍Sites Optimizer如何分析前端和后端实施以检测冗余脚本、大量DOM结构和Core Web Vitals瓶颈。 了解如何在AEM中直接应用其建议以减少技术债务并提高性能。 在圣何塞现场录制。
