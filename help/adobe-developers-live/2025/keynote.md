# 主旨：建立代理网站

通过本主题演讲，了解AI如何转变Adobe Developers Live 2025的数字体验。 了解Adobe对代理Web的愿景，该愿景包含对AEM代理、Content AI基础服务、Brand Concierge等的洞察。 此活动汇集了众多专家，共同讨论面向全球开发人员和架构师的创新解决方案。

>[!VIDEO](https://video.tv.adobe.com/v/3477589/?learn=on&enablevpops)

在圣何塞现场录制时，全主题演讲将特别关注Loni Stark、David Nuescheler、Mark Szulc、Shankari Panchapakesan、Cecily Liu和Deck Reyes，他们将揭示Adobe对代理Web的愿景，包括AEM Agents、Content AI Foundation Services、Brand Concierge、Sites Optimizer和LLM Optimizer。

特别感谢我们的赞助商Algolia和Ensemble对Adobe Developers Live 2025的支持。
