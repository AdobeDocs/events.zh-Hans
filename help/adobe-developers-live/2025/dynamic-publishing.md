# Edge中的BYOM和App Builder动态发布

使用自带标记和Adobe App Builder探索边缘的动态发布。 了解这种创新方法如何在不使用传统CMS创作的情况下直接在边缘提供动态内容。 了解如何获取外部数据并将其转换为语义HTML，确保快速且样式化的交付。 此会话提供了对现实实施情况的见解，从而为项目解锁新的发布模型。 继续有关Experience League的对话，并发现即将发生的事件。

>[!VIDEO](https://video.tv.adobe.com/v/3477716/?learn=on&enablevpops)

本课程 — Edge动态发布与BYOM和App Builder — 重点介绍Adobe高级技术架构师Lars Auffarth，并预先录制了Adobe Developers Live 2025课程。 Lars演示了如何使用您自己的标记和Adobe App Builder在边缘直接提供动态内容，而无需传统的CMS发布。 了解此架构如何结合创作内容和动态内容，通过App Builder操作获取外部数据，并以边缘速度呈现语义HTML — 为现实世界项目解锁新的发布模型。
