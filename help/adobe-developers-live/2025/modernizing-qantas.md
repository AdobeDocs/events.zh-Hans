# 使用Edge Delivery Services (EDS)实现Qantas.com的现代化

了解澳航如何使用Adobe Experience Manager Edge Delivery Services对其网站进行改版。 此转换将发布时间从几周减少到几分钟，简化了开发，并通过更简洁的语义标记改进了SEO。 了解从内部部署向边缘架构的转变，以及它如何通过基于JavaScript的工作流扩展人才库。 深入了解这种现代方法如何提高创作速度并支持公司治理目标。 加入对话并探索即将举办的Adobe活动，以获得更深入的见解。

>[!VIDEO](https://video.tv.adobe.com/v/3477681/?learn=on&enablevpops)

这场与Edge Delivery Services的炉边聊天 — 实现Qantas.com现代化 — 邀请了Accenture Song解决方案架构师Tim Lavelle和Adobe高级产品专家Glen Lawson从悉尼现场录制。 他们讨论了澳航的主要重建项目以及向Adobe Experience Manager Edge Delivery Services的转移，重点介绍了从内部部署向边缘架构的转移如何将发布时间从几周缩短到几分钟。 了解这种现代方法如何简化开发，如何使用基于JavaScript的工作流扩展人才库，以及通过更简洁的语义标记和通用编辑器改进创作速度和SEO。

特别感谢我们的赞助商Algolia和Ensemble对Adobe Developers Live 2025的支持。
