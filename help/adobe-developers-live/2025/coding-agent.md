# AI编码之争：好、坏、丑

了解AI编码代理在真实的Edge Delivery Services项目中的执行情况。 了解他们的优势、挑战及其对开发人员体验的影响。 来自Adobe Developers Live 2025的这个会议展示了实际自动化与“vibe coding”的对比，重点介绍了AI的优势和挣扎之处。 参加拉尔斯·特里洛夫的基因工程测试，比较领先的人工智能编码代理，揭示数字体验的未来。

>[!VIDEO](https://video.tv.adobe.com/v/3477619/?learn=on&enablevpops)

这个讲座 — Coding Agent Showdown： The Good， the Bad， and the Ughtly — 对代理工程进行了测试。 Lars Trieloff对真实Edge Delivery Services项目中的领先AI编码代理进行了比较，揭示了开发人员体验与代理体验之间的关联。 看看人工智能在何处闪耀，在何处挣扎，以及实际自动化与“视频编码”之间的区别。 在圣何塞现场录制。
