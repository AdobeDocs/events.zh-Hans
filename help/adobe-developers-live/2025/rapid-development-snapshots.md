# 冻结、恢复、创新：快速开发环境的动手快照用于云开发

了解Adobe Experience Manager Cloud Service中的快速开发环境(RDE)如何支持云原生的swift迭代。 了解快照功能如何使开发人员能够冻结、还原和重新使用完整的RDE状态。 这使您能够使用Adobe I/O CLI RDE插件简化测试、即时回滚并加快开发周期。 提高云开发效率并与公司治理政策保持一致。

>[!VIDEO](https://video.tv.adobe.com/v/3477705/?learn=on&enablevpops)

本课程 — 冻结、恢复、创新：快速开发环境动手快照用于云开发 — 专题介绍Adobe软件开发工程师Remo Liechti，并为Adobe Developers Live 2025年预先录制。 Remo演示了快速开发环境(RDE)如何在Adobe Experience Manager Cloud Service中实现快速的云原生迭代，以及新的快照功能如何让开发人员冻结、还原和重新使用完整的RDE状态。 了解如何使用AEM CLI RDE插件在Adobe I/O Cloud中简化测试、即时回退和加快开发周期。
