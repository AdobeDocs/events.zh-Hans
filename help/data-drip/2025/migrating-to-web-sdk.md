# 数据流：从旧系统迁移到Web SDK的好处

参加此数据滴流网络研讨会，Adobe客户技术顾问Arindam Agarwal将指导我们了解Adobe Web Software Development Kit的系统优势。 Adobe Web SDK是一种客户端解决方案，它通过Adobe Experience Platform Edge Network利用其服务，创造了持续的竞争优势，降低了运营成本，并与旧版库相比提高了性能。 Arindam将深入介绍Web SDK，以及它的功能和优势，例如： — 易于实施 — 简化的集成 — 更智能的跟踪支持 — 责任降低 — 事件端转发 — 网站流量流

## 解锁Adobe Web SDK的强大功能

了解迁移到Adobe Web SDK如何转变数字数据收集和个性化。 这种现代方法可简化您的组织的工作流程和对未来的证明。

* **统一数据收集**&#x200B;将多个库替换为单个轻量级SDK，从而减轻页面重量并减少维护。
* **Edge Network集成**&#x200B;通过Adobe的分布式服务器路由所有数据，以便高效地传送到分析、个性化和第三方平台。
* **性能和合规性**&#x200B;享受更快的网站、改进的治理和内置的隐私功能，如同意管理。
* **简化的实施**&#x200B;使用清晰的规划、架构映射和强大的调试工具进行迁移。

利用这些功能可实现更干净的数据、更强的合规性和可扩展的数字体验。

## 统一数据收集解释

Adobe Web SDK将多个跟踪库(Analytics、Target、Audience Manager)整合到一个统一的alloy.js脚本中。 此单个SDK：

* 在一次调用中将所有数据发送到边缘网络，从而减少网络请求和页面加载时间。
* 处理数据收集和个性化，而无需单独的请求。
* 与Adobe的边缘网络集成，该网络将数据路由到配置的解决方案和第三方目标。
* 支持标准化的体验数据模型(XDM)架构，以实现各平台间的数据一致。

此方法简化了维护，提高了性能，并实现了团队之间的无缝协作。
