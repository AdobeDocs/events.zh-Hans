# 2026.03.0 AEM发行版概述视频

>[!VIDEO](https://video.tv.adobe.com/v/3483070/?captions=chi_hans&learn=on&enablevpops)

此版本中的功能重点针对Experience Manager Sites、Forms、Assets和Cloud Foundation创新，包括以下内容：

* 站点
   * Experience Production Agent更新和启动支持

* 表单
   * 支持IC编辑器的工作流
   * 对核心组件的主题编辑器支持
   * Forms的版本控制支持

* 资产
   * AEM内容顾问 — GA与Adobe应用程序
   * Microsoft Office连接器(PowerPoint、Word、Excel)

* 云基础
   * 治理代理：品牌管理
   * 触屏UI上的审核权限
   * OIDC注销功能
   * 暂停维护更新

* Cloud Manager
   * Cloud Manager MCP服务器
