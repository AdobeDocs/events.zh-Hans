# AEM as a Cloud Service 2023.9.0发行版更新

此版本中的功能主要针对Experience Manager Assets和Cloud Service创新，包括以下内容：

**发布功能**

* 资产
   * 适用于Adobe Express的AEM Assets加载项
   * 将Anywhere与资产选择器SDK集成

* 云基础
   * 内容传输工具3.0
   * 在Cloud Manager中取消管道
