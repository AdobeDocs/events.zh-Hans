# 2024.9.0 AEM发行版概述视频

此版本中的功能主要针对Experience Manager Assets和Cloud Service创新，包括以下内容：

## AEM Assets

* 支持收藏集的资产选择器增强功能&#x200B;
* Assets Content Hub — 支持DRM（已过期和已授予许可的资源）&#x200B;。

## AEM云服务

* Cloud Manager支持Edge Delivery&#x200B;
* Edge端包含(ESI)&#x200B;
* 基本身份验证&#x200B;
* 交互式AEM Developer Console
