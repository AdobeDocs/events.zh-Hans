# 2024.7.0 AEM发行版概述视频

此版本中的功能主要针对Experience Manager Assets和Cloud Service创新，包括以下内容：

**AEM Assets**

* AEM Assets中的Content Credentials&#x200B;
* 使用Dynamic Media为视频提供AI支持的字幕&#x200B;
* 用于上传的资源选择器增强功能&#x200B;

**Cloud Manager**

* 客户管理的CDN凭据的自助配置&#x200B;。
* CDN清除&#x200B;
* 流量过滤器规则警报&#x200B;。
