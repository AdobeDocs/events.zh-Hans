# 2024.10.0 AEM发行版概述视频

此版本中的功能主要针对Experience Manager Assets和Cloud Service创新，包括以下内容：

AEM Assets：
* AEM Assets与Assets Ultimate和Asset Prime的新定价和包装

AEM Cloud Service：
* 将日志转发到更多目标
* 适用于企业用户的无管道URL重定向&#x200B;
* RDE支持配置管道YAML&#x200B;
* 交互式AEM Developer Console
