# 2025.01.0 AEM发行版概述视频

此版本中的功能重点针对Experience Manager Sites、Forms、Assets和Cloud Foundation创新，包括以下内容：

* AEM Sites
   * 内容片段注释

* AEM Forms
   * 与AEM Forms的交互式通信

* AEM Assets
   * Dynamic Media模板
   * Dynamic Media投放报告

* AEM Cloud Foundation
   * 使用Java 17或21构建
