# 2025.04.0 AEM发行版概述视频

此版本中的功能重点针对Experience Manager Sites、Forms、Assets和Cloud Foundation创新，包括以下内容：

## AEM Sites：

* 内容模型管理员UI
* 内容片段启动项UI和API
* Edge Delivery Services&#x200B;和Commerce样板

## AEM CLOUD基础

* 支持经过身份验证的基于OpenAPI的API

## AEM Forms：

* 对基于Edge Delivery的Forms的调频支持
* 在规则编辑器UI中突出显示损坏的规则
