# 一次投放，尽情体验：使用Adobe Dynamic Media扩展全渠道体验

您是否想知道，如何简化AEM Assets中图像和视频的转换和优化？

加入Adobe Experience Manager Champion Vengadesh Shanmugavelu，了解有关在Adobe Experience Manager Assets中使用Dynamic Media的见解。 他将介绍扩展数字内容创建的关键要素，包括：

>[!VIDEO](https://video.tv.adobe.com/v/3481555/?captions=chi_hans&learn=on&enablevpops)

* Adobe Experience Manager Dynamic Media如何使用配置文件设置、智能演绎版和实时优化来启用智能媒体分发。
* 如何通过AEM核心组件激活Dynamic Media，并使用Dynamic Media Open API在整个企业内扩展其功能。

无论您是营销人员、架构师还是开发人员，此网络研讨会都将向您展示如何一次性交付，以及如何通过性能、一致性和控制力随时随地扩展。

Vengadesh Shanmugavelu是Publicis Sapient高级工程经理。 他在内容管理解决方案体系结构、设计、开发、实施、培训和支持AEM和Interwoven产品套件方面拥有15年以上的经验，在AEM和Adobe Marketing Cloud解决方案方面拥有7年以上的经验。 他是Adobe认证专家（担任AEM的架构师和首席开发人员）、Adobe认证专家（担任Adobe Target架构师）和Adobe认证的Experience Platform工程师。 他自2022年以来一直是AEM冠军。
