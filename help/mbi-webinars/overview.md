# MBI网络研讨会

MBI网络研讨会系列是面向MBI客户的定期活动。 这些网络研讨会涵盖了充分利用MBI的各种主题和最佳实践。

来自所有地区的与会者都可现场参加活动，以更好地了解产品并获得关于MBI功能的问题的直接解答。

这些主题会偶尔重复出现，以适应客户可能提出的任何新问题。
