# 在 MBI 中管理数据集

直接从 Adobe Commerce 产品团队了解 MBI 数据仓库管理器的一些强大功能。超越基本的报告构建，了解如何利用您的数据做更多事情。

在這場錄製的網路研討會中，您將瞭解如何透過聯結Adobe Commerce、Google E-commerce和Google AdWords的資料，來計算廣告促銷活動的投資報酬率。
