# 您的Marketo路线图

Marketo可定制且可扩展，因此带来了大量机会。 但与此同时，计划不周或缺乏计划，可能会让你面临一个规模糟糕的Marketo实例和大量科技债。

将您的Marketo实例视为产品很有帮助。 您有客户，并且需要将您的实例开发成一种让客户满意的产品。 但是，匆忙进行生命周期营销或反补贴税可能收效不佳。 您还有一个产品路线图，它是您的工作计划，指导您如何实现目标。
