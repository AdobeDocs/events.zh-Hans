# 您的Marketo路线图

Marketo可自定义且可扩展，因此带来了大量机会。 但与此同时，糟糕的规划或者缺乏规划，可能会让你陷入规模糟糕的Marketo实例和大量科技债务。

将您的Marketo实例视为产品会有所帮助。 您有客户，并且需要将您的实例开发成一种让客户满意的产品。 但是，急忙推出生命周期营销或ABM或许收效不佳。 您还有一个产品路线图，这是您的工作计划，指导您如何实现目标。
