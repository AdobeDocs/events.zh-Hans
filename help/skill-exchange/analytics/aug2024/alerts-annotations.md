# 自动化感知 — 使用警报和注释来防止意外发生

想要立即发现问题而不用盯着图表看吗？ 想要在分析旧数据时考虑异常情况，而不记住所发生的情况？ 使用警报和注释可让Adobe Analytics意识到性能的重要更改并随时提醒您。

在本课程中，您将了解如何：

1. 使用警报可立即自动了解性能更改
2. 使用注释来显示重要事件发生时间
3. 自定义这两者，以便仅通知受影响的组
