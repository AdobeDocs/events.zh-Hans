# 技能交流活动2023年8月 — AEM开发人员跟踪 — 迁移到AEM Cloud Service的战略与战术提示

* 更好地了解迁移的价值并能够在内部销售迁移的价值
* 管理迁移风险和成本并高效利用现有工作的方法
* 在任务较大时逐步完成迁移的技术策略
