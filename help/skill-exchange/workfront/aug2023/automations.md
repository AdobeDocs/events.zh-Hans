# 目标驱动自动化

借助自动化，您的用户可以将注意力集中在需要独特技能和专业知识的工作上，而不是让他们疲于应付的例行而重复的任务，从而释放出巨大的效率潜力。

在本次研讨会中，来自Zimmerman Advertising的Jennafer Higgs、Jonathan Cabre和Malcom Benites将分享经证明可创建更高效工作流的自动化示例。

它们将共享Fusion和非Fusion自动化，包括：

·如何自动关闭项目
·子模板生成
·综合经批准资产审阅
